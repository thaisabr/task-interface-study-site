RQ3: Is TestI a better code change predictor than RandomI?


TestI is a more precise predictor than RandomI

In the best case, TestI predicts as many changed files as a random predictor

Wilcoxon Signed-Rank test applied to precision and recall measures of ITest and IRandom
Hypothesis pvalue r
TestI-NF precision = RandomI-NF precision 3.4e-74 0.85
TestI-NF recall = RandomI-NF recall 2.4e-43 0.64
TestI-CF precision = RandomI-CF precision 1.0e-54 0.72
TestI-CF recall = RandomI-CF recall 1.9e-01 0.06
